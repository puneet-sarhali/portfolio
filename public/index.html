API HUB

My first front-end project, In which I designed and built a prototype for an API Marketplace using HTML and CSS.

WEBSITE GITHUB

A little Overview:

We were assigned to create a mock front-end for an online shop using HTML and CSS. Me and my partner, Gaurav Gupta came up with the idea to create an API marketplace. My contribution consisted of designing and implementing landing page and individual product pages.

Problems I encountered:

  1. first I was copying and pasting hex codes for each of the colors that I was using, this made any change to color pallete a very manual task.

  2. In terms of design, We couldnâ€™t use the structure of image based product listings because a developer who is looking for an API will not base his decision on a picture and a price tag.

Solutions that worked:

  1. My experience with other languages helped me find out that there are variables available for use is CSS aswell.

  2. I decided to create the listing such that it shows more information about the API than a large image. Similarly, Instead of a single price tag a developer wants a choice between tiers which fits to his use of the API. So, I used CSS grids to create responsive listings and pricing models for the website.

What i learned

  1. Maintaining a large CSS file is difficult. Therefore, modularized code with sass is beneficial.

  2. Absolute positions and fixed values makes it very difficult to make the website accessible.
